## 🔍 What is Dark Files?

**Dark Files** is an immersive, AI-powered detective game where every case is uniquely generated just for you. Step into the shoes of a detective, interrogate suspects, uncover hidden clues, and solve mysteries — all powered by a real AI Game Master that responds intelligently to your every move.

No two cases are ever the same. No answers are handed to you. Every deduction is yours to make.

## ⚙️ Setup & Installation

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/demonxyonko/dark-files.git
cd dark-files
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Get Your Free Groq API Key
1. Go to **[console.groq.com](https://console.groq.com)**
2. Sign up with Google — completely free
3. Click **API Keys** → **Create API Key**
4. Copy your key (starts with `gsk_`)

### 4️⃣ Run Locally
```bash
streamlit run dark_files_web.py
```

### 5️⃣ Open in Browser
```
http://localhost:8501
```

## 🕹 How to Play

```
1. 🔑  Enter your Groq API key on the start screen
2. 🕯  Click "Open First Case" to receive your case briefing
3. 🔍  Investigate by typing naturally:
        → "Tell me about the suspects"
        → "Examine the crime scene"
        → "Interrogate [suspect name]"
        → "Request more clues"
4. 🧩  Piece together the evidence and form your theory
5. ⚖️  Submit your final answer:
        → "My final answer is: [suspect] because..."
6. 📋  Receive your verdict and full case solution
```
